# Support Ticket Triage — OpenEnv 🚀

**Team:** Fresh Tensors (Vikash S)
**Event:** Meta PyTorch OpenEnv Hackathon x Scaler School of Technology

## Project Abstract

This repository contains a production-ready OpenEnv simulation for training AI agents in frontline customer support triage.

## Endpoints

- `POST /reset` — Start a new episode
- `POST /step` — Submit an action
- `GET /state` — Get current state
- `GET /tasks` — List available tasks
- `GET /grader` — Get scores
- `GET /health` — Health check
